An application that reads PDB files
To build a map of the peptide space

Information

In order to get more information about peptide and how they operate in the 3D space, this application was made. It reads PDB files and analyses and visualizes oligopeptides with the given length!

Upload Protein

Click here to upload a protein file (PDB) and start the process. The page will tell you more about what to do exactly!

Code and Papers

The code for this project is on github. Want to know more about this project topic? Structural Alphabet: paper 1 and paper2!

Workflow

Start

Retrieve the coordinates and other information like peptide from the PDB file

Use the XSSP api to get the secondary structures

Split the coordinates on breaks/chains

Get the internal distance between the coordinates

Get the cov matrix of the distances

Perform PCA on the cov matrix and retrieve the eigenvectors

Perform matrix multiplication on the distance and eigenvectors

End
